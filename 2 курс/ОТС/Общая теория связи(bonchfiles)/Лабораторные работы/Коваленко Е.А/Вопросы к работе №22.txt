	Вопросы к работе №22. «Согласованная фильтрация сигналов известной формы»

1. Выберите подходящие осциллограммы для реакции фильтра, согласованного с прямоугольным видеоимпульсом, на «свой» сигнал.
2. Выберите подходящие осциллограммы для реакции фильтра, согласованного с прямоугольным радиоимпульсом, на «свой» сигнал.
3. Выберите подходящие осциллограммы для реакции фильтра, согласованного с кодом Баркера, на «свой» сигнал.
4. Выберите подходящие импульсные характеристики фильтров, согласованных с прямоугольным видеоимпульсом.
5. Выберите подходящие АЧХ фильтра, согласованного с прямоугольным видеоимпульсом.
6. Выберите подходящие АЧХ фильтра, согласованного с прямоугольным радиоимпульсом.
7. Определите число периодов ВЧ заполнения прямоугольного радиоимпульса по АЧХ согласованного с ним фильтра.
8. Определите длительность прямоугольного видеоимпульса по АЧХ согласованного с ним фильтра.
9. Определите число периодов ВЧ заполнения прямоугольного радиоимпульса по АЧХ согласованного с ним фильтра.
10. Выберите подходящие импульсные характеристики фильтров, согласованных с прямоугольным радиоимпульсом.
11. Выберите подходящие импульсные характеристики фильтров, согласованных с кодом Баркера.
12. Определите момент максимума реакции СФ на "свой сигнал" по импульсной характеристике фильтра. 
13. Какие фильтры называют согласованными с сигналами? 
14. Выберите подходящие выражения для передаточной функции - H(jw), АЧХ - H(w) и ФЧХ - ф(w) согласованного фильтра. 
15. Выберите подходящее выражение для формы отклика согласованного фильтра на «свой» сигнал.
16. Определите длительность реакции СФ (в мс) на «свой» сигнал по импульсной характеристике фильтра. 
17. Выберите подходящее выражение для отношения с/ш на выходе согласованного фильтра?
18. Выберите подходящие выражения для передаточной функции, АЧХ и ФЧХ согласованного фильтра при небелом шуме на его входе?
