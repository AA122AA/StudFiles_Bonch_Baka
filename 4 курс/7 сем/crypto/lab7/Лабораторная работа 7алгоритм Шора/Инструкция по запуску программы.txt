1. Попробовать запустить QuantumSimulator.exe. В случае неудачи или при его некорректной работе, перейти к п.2. Если компьютер, на котором запускается программа, имеет выход в сеть, будет предложено установить язык java с открытием в браузере ссылки, упомянутой в п.3.
2. Проверить, установлен ли на компьютере язык java. Для этого зайдите в командную строку cmd через Пуск -> Выполнить -> cmd или по нажатию на комбинацию клавиш Win+R -> cmd. Win - та клавиша со значком операционной системы Windows. После того, как запустится командная строка, введите «java –version» без кавычек: если программа установлена, то будет отображена её версия, если нет, то об этом будет сообщено. Если язык java установлен в системе, перейдите к п.4., если  нет, то к п.3.
3. JAR-файлы нельзя запустить, если на компьютере нет Java. Перейдите на веб-сайт Java по адресу https://www.java.com/ru/download/ и нажмите «Загрузить Java бесплатно», или запустите файл «chromeinstall-8u211.exe», располагающийся в той же папке, что и эта инструкция, а затем установите Java. Возможно, придется перезагрузить компьютер.
4. Дважды щелкните по JAR-файлу «QuantumSimulator.jar». Если он является исполняемым и на компьютере есть Java, файл должен открыться. Если он определяется имеющимися на компьютере программами как архив, то щёлкните правой кнопкой мыши на файле, выберите «Открыть с помощью» > «Java (TM)» > «ОК». Еили откроется всплывающее окно с запросом, какую программу использовать, чтобы открыть файл, в этом случае нажмите «Java (TM)» > «ОК».
5. Повторить действия 1-4 применительно к программе CountFrac.exe

Перед тем, как начнете пользоваться программой, убедительная просьба прочитать прилагающуюся к ней инструкцию (находящуюся в той же папке, что и эта) для упрощения взаимодействия с интерфейсом!
